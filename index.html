Решение задач:

1.В переменных a и b хранятся числа. Написать программу, которая выводит в консоль сумму этих чисел.

2.В переменных a и b хранятся числа. Написать программу, которая выводит в консоль произведение чисел.

3.В двух переменных хранятся строки символов. Написать программу, которая выведет в консоль суммарное количество символов в обоих строках.

4. Написать программу, которая запрашивает у пользователя ввод трёхзначного числа, а потом выводит в консоль сумму цифр введённого числа.

5. В переменных a и b хранятся числа. Вывести в консоль наибольшее из них.

6. Запросить у пользователя ввод числа от 1 до 12. Вывести в консоль название месяца, соответствующее этому числу (1 — январь, 2 — февраль и т.д.)

7. В переменных circle и square хранятся площади круга и квадрата соответственно. Написать программу, которая определяет, поместится ли круг в квадрат.

8. Вывести в консоль сумму всех целых чисел от 50 до 100

9. Вывести в консоль таблицу умножения на 7

10. Запросить у пользователя ввод числа N. Вывести в консоль среднее арифметическое всех нечётных чисел от 1 до N.

11. Создайте объект user, содержащий поле name со значением ‘John’.

1.Запросить у пользователя ввод числа. Записать введенное значение в поле age объекта user.

2.Создать копию объекта user с именем admin. Добавить новому объекту поле role со значением ‘admin’.

Записать все значения полей объекта admin в отдельные переменные. Имена переменных должны совпадать с названиями полей.

12. Создайте массив целых чисел из десяти элементов. Выведите в консоль сумму всех элементов массива.

Создайте новый массив на основе исходного, в котором

каждый элемент будет вдвое больше элемента исходного массива с таким же индексом.

Найдите и выведите в консоль наибольший и наименьший элементы исходного массива.

13. Напишите функцию diff, которая получает в качестве параметров 2 числа и возвращает разницу между наибольшим и наименьшим.

14. Напишите функцию isWord, которая принимает на вход текстовую строку.Функция возвращает true, если строка состоит из одного слова и false, если из нескольких.

15. Напишите функцию pow(a, x), которая вернёт значение числа a, возведённого в степень x.

17. Запросите у пользователя дату в формате ДД.ММ.ГГГГ.Напишите программу, выводящую день недели по введённой дате

18. Написать программу, которая выводит в консоль количество минут, прошедшее с начала сегодняшнего дня.

19. В двух переменных хранятся даты рождения двух пользователей в формате ДД.ММ.ГГГГ.Написать программу, которая определяет более молодого пользователя.

20. Даны длины трёх сторон треугольника.Определить, является ли треугольник прямоугольным.

21. Пользователь вводит число R.Написать программу, которая выведет в консоль длину окружности и площадь круга с радиусом R.

22. Пользователь вводит числа a, b и c.Написать программу, выводящую корни квадратного уравнения с коэффициентами a, b и c.

23. Пользователь вводит текстовую строку.Определить с помощью регулярного выражения, является ли введённая строка: 1.Датой. 2.Адресом электронной почты. 3. * Номером телефона.
